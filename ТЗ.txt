
ТЕХНИЧЕСКОЕ ЗАДАНИЕ: TickTask

1. Зависимости: все библиотеки перечислены в requirements.txt

2. Фронтенд: используется Bootstrap для адаптивного интерфейса

3. ORM: реализованы модели пользователь, задача, категория, статистика

4. Аутентификация: реализована регистрация и авторизация пользователей

5. Файлы: реализована загрузка и использование файлов для импорта и экспорта задач

6. API: используется стороннее API для работы искусственного интеллекта

7. Хранение данных: данные хранятся в базе данных

8. Хостинг: приложение развёрнуто на хостинге

9. Быстрый ввод задач: пользователь может вводить задачи текстом

10. ИИ-парсинг: искусственный интеллект автоматически извлекает из введённой задачи дату, время, категорию и предполагаемую длительность

11. Сортировка по важности: искусственный интеллект автоматически сортирует задачи по важности

12. Умное планирование: искусственный интеллект составляет расписание на основе задач

13. Напоминания: система напоминает пользователю о незавершённых делах

14. Контроль задач: реализован контроль незавершённых дел

15. Статистика: ведётся учёт и статистика продуктивности пользователя

16. Интеграция: реализована интеграция с различными сторонними сервисами

17. Мотивация: система формирует мотивационные отчёты в конце недели
